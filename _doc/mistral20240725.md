Here's a simple Python function that takes a single argument (a string) 
and returns the count of each occurrence of a specific character within 
the given string:
```python
def char_count(string, char):
    result = {}
    for letter in string:
        if letter == char:
            if char not in result:
                result[char] = 1
            else:
```
